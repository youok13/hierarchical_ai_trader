# Hierarchical AI Trader

## Overview

This project implements a **Hierarchical AI Trading System** designed to analyze financial market data and make intelligent trading decisions using machine learning techniques.

---

## Features

* Data-driven trading strategy
* Machine learning-based prediction
* Modular Python architecture
* Scalable and extensible design

---

## Dataset

The dataset used in this project is in CSV format and contains historical financial data.

Example:

* Stock prices
* Market indicators
* Time-series data

---

## Project Structure

```
hierarchical_ai_trader/
│
├── data/
│   └── dataset.csv
│
├── src/
│   ├── main.py
│   ├── model.py
│   └── utils.py
│
├── requirements.txt
└── README.md
```

---

## Installation

1. Clone the repository:

```
git clone https://github.com/youok13/hierarchical_ai_trader.git
```

2. Navigate to the project:

```
cd hierarchical_ai_trader
```

3. Install dependencies:

```
pip install -r requirements.txt
```

---

## Usage

Run the main script:

```
python main.py
```
